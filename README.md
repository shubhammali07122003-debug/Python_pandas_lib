Learning pandas:
Pandas + SQLAlchemy Data Analysis Project

This project demonstrates practical use of Python Pandas for data cleaning, transformation, and analysis, along with SQLAlchemy for connecting and interacting with a PostgreSQL database.

Tools Used =

Python

Pandas

SQLAlchemy

PostgreSQL

Jupyter Notebook

Key Skills Applied :

Loading data into Pandas DataFrames

Data cleaning & preprocessing

Handling missing values

GroupBy & aggregations

Creating calculated columns

Connecting to PostgreSQL using SQLAlchemy

Reading/Writing data between Pandas and Database

🎯 Objective

To understand real-world data workflow by integrating Pandas with a relational database using SQLAlchemy.
